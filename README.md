# Student Depression Analysis Dashboard

## Overview
This project analyzes factors contributing to student depression using data visualization in Tableau.

## Dashboard
Interactive dashboard:
https://public.tableau.com/app/profile/saket.gupta2331/viz/StudentDepressionAnalysis_17751520425750/SudentCountAnalysis

## Dataset
The dataset contains student lifestyle, academic pressure, sleep duration, financial stress, and depression indicators.

## Tools Used
- Tableau
- SQL
- Data Visualization
